Forecasting Oil and Gold Prices

Complete

Forecasted oil and gold prices by analyzing time series correlation with stock indices (NASDAQ, DJIA). Selected optimal models using AIC/BIC/MSE criteria—ARDL(1,1) for oil, AR(1,1) for gold. Ran VAR models to test interaction effects. Applied CCF, IRF, CUSUM, and FEVD plots for multivariate time series analysis.

Time Series ARIMA VAR Models R Econometrics

Additional Projects

Predicting Taiwanese Business Bankruptcy

Complete

Developed Random Forest classification model to predict bankruptcy likelihood for 1000+ startup companies. Compared logistic regression with lasso and random forest models for feature selection. Identified model imbalance issues and corrected using bagged pooling methods. Discovered that growth rate metrics (Net Value Growth Rate, Persistent EPS, Interest Bearing Debt) are stronger bankruptcy predictors than absolute financial values.

Python Machine Learning Random Forest Logistic Regression Feature Selection

Optimizing Portfolios with Statistical Models

Complete

Generated efficient frontiers for portfolios of 30 stocks across multiple industries using mutual fund theorem. Estimated frontiers using Single Index Model (SIM), constant correlation model, and multi-group model. Applied Jensen differential and Sharpe ratio to analyze performance. Used Vasicek and Blume methods to forecast 2022-2025 values with PRESS statistics for model selection.

Statistical Modeling Portfolio Theory R CAPM Time Series

EY Bank Analysis and Visualization

Complete

Used Power BI and Tableau to visualize comprehensive dashboards analyzing debt securities distribution across bank sizes. Transformed massive CSV/Excel datasets using Power Query and calculated interest rate metrics with table formulas. Tracked trends in HTM (Held-to-Maturity) and AFS (Available-for-Sale) bonds over time.

Power BI Tableau Power Query Excel Data Visualization

Impacts of Capital Investment on Unemployment

Complete

Analyzed panel data (1970-1986) examining relationships between public/private capital, gross state product, and unemployment rates. Identified California and New York GSP growth 10x higher than peers using pooling/fixed/random effects models. Found Texas severely underperforming despite high mean GSP, attributing results to population expansion and technological development.

Panel Data Econometrics R Regression Analysis Economic Analysis
